# 🔑 Key Insights

- Total cargo increased by **2.4%**
- Incoming cargo decreased
- Outgoing cargo increased
- Tanker cargo increased significantly
- Mumbai handled highest cargo

---

# 📍 Port Insights

- Mumbai → Highest volume
- Madras → Decline
- Kochi → Growth
- Kolkata → Lowest activity
